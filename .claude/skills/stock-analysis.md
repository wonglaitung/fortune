# 股票分析技能

当用户询问特定股票的买卖建议时，全面查询综合分析报告并提供专业回答。

## 触发条件

用户询问以下类型的问题时触发此技能：
- "今天买XXX股票好不好"
- "XXX股票值得买吗"
- "XXX现在可以买入吗"
- "XXX股票分析"
- "帮我分析XXX股票"
- "XXX股票怎么样"
- "应该卖出XXX吗"
- "对XXX怎么看"
- "对XXX基金怎么看"
- "怎么看XXX"
- "XXX如何"
- "XXX表现如何"

## 执行步骤

### 1. 确定查询日期

首先获取最近的交易日日期：
- 如果今天是交易日，使用今天的日期
- 如果今天是周末或假期，使用最近的交易日（通常是周五）

### 2. 读取报告文件

读取 `output/comprehensive_reports/YYYY-MM-DD.md` 文件。

- 常见简称（如"农行"、"腾讯"）

### 4. 提取关键信息

从报告中提取以下**全部相关信息**：

#### A. 综合建议部分
- 强烈买入信号（股票列表及详细建议）
- 买入信号
- 持有/观望
- 卖出信号

#### B. 个股关键数据（从预测表格提取）
- CatBoost 上涨概率（1天/5天/20天预测）
- 三周期模式（如"下跌中继⭐(001)"）
- 交易建议（谨慎做多/分批建仓/止损减仓）
- 历史胜率
- 传导模式验证
- 筹码阻力（低/中/高）
- 风险得分/回报得分/综合得分
- 风险建议（推荐/观察/暂缓/优选）
- 网络洞察（社区归属、枢纽类型）

#### C. 市场环境信息
- 恒生指数当前价格和涨跌幅
- 市场状态（牛市/震荡偏涨/震荡市/震荡偏跌/熊市）
- 市场状态持续时间及稳定性
- HMM状态转换概率
- VIX指数
- 市场情绪（正常/谨慎/暂停）

#### D. 异常检测信息
- 是否有异常检测警报
- 异常类型（成交量异常/Isolation Forest异常）
- 异常原因（超买/超卖/量比异常等）
- 大模型异常分析结论

#### E. 板块分析
- 所属板块名称
- 板块5日涨跌幅排名
- 板块类型（周期/防御）
- 板块龙头股

#### F. 大模型建议
- 短期建议（买入/观察/卖出）
- 中期建议（买入/观察/卖出）
- 建议仓位
- 具体操作时机

#### G. 技术指标详情
- RSI（超买>70/超卖<30）
- MACD（金叉/死叉）
- 布林带位置
- 均线排列（多头/空头/震荡）
- 支撑位/阻力位
- ATR波动率

#### H. 网络洞察预警
- 模块度（市场分化程度）
- 是否为桥梁股
- 社区归属

#### I. 波动率网络密度预警
- 当前密度值
- 预警状态（正常/关注/预警）

#### J. 股息信息
- 即将除净日期
- 分红方案

### 5. 综合分析框架

按以下逻辑综合判断：

**硬约束检查**：
1. CatBoost 20天上涨概率 ≤ 0.50 → **禁止买入**
2. CatBoost 20天上涨概率 ≥ 0.60 → 高置信度，可考虑买入
3. CatBoost 20天上涨概率 0.50-0.60 → 中等置信度，需其他信号确认

**市场环境检查**：
1. 市场状态为"熊市" → 提高阈值至0.70
2. 市场状态为"震荡市" → 提高阈值至0.65
3. 市场状态稳定性<5天 → 提示风险，建议降低仓位

**异常检测检查**：
1. 有超买异常（RSI>70）→ 提示短期回调风险
2. 有超卖异常（RSI<30）→ 提示可能存在反弹机会
3. 有成交量异常 → 提示资金异动

**网络洞察检查**：
1. 模块度<0.20 → 市场同涨同跌，降低仓位30%
2. 为桥梁股 → 提示跨社区风险传导
3. 波动率密度预警 → 按预警级别调整仓位

### 7. 特殊情况处理

**找不到报告文件**：
- 提示用户"当前没有最新的分析报告，请等待今日收市后的综合分析"

**找不到该股票**：
- 提示用户"XXX不在当前的自选股列表中，无法提供分析"

**报告日期较早**：
- 提示用户"最新报告日期为 YYYY-MM-DD，数据可能已过时，请谨慎参考"

**存在异常检测警报**：
- 在回答中醒目标注异常信息，提醒用户注意风险

**市场环境恶劣**：
- 当市场状态为熊市或状态不稳定时，在回答开头添加风险警示

## 注意事项

1. **硬性约束**：CatBoost概率 ≤ 0.50 时，绝对禁止推荐买入
2. **风险控制**：所有建议必须包含止损位，最大亏损控制在-8%以内
3. **数据时效**：报告基于收市后数据，盘中查询时需提示数据可能不完整
4. **异常优先**：如有异常检测警报，优先展示并提示风险
5. **市场环境**：根据市场状态动态调整建议的保守程度
6. **网络洞察**：模块度低时提示系统性风险，建议降低仓位
7. **免责声明**：每次回答末尾添加"以上建议仅供参考，不构成投资建议，投资有风险，决策需谨慎"
