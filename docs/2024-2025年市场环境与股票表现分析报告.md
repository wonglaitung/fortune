# 2024-2025年市场环境与股票表现分析报告

## 执行摘要

本报告分析了2024-2025年期间港股市场环境（牛市、熊市、震荡市）的分布情况，以及28只股票在不同市场环境下的表现差异。通过分析每只股票与市场环境的关联性，为投资策略制定提供数据支持。

**关键发现**：
- 2024-2025年市场整体表现良好，年化收益率约+1.24%
- 牛市月份占37.5%，熊市月份占25.0%，震荡市月份占37.5%
- 19只股票为高市场关联性（牛市表现优异，熊市表现较差）
- 7只股票为低市场关联性（牛熊市表现差异小，相对稳定）
- 工商银行（1398.HK）的市场关联性仅0.05，几乎不受市场环境影响

### 3.2 市场关联性定义

**市场关联性** = (牛市收益率 - 熊市收益率) / |牛市收益率 + 熊市收益率|

- **高市场关联性（> 0.3）**：牛熊市表现差异大，随市场涨跌
- **中等市场关联性（0.1-0.3）**：市场影响适中
- **低市场关联性（< 0.1）**：牛熊市表现差异小，相对独立
- **负市场关联性（< 0）**：逆势表现，熊市表现优于牛市

| 指标 | 数值 |
|------|------|

| 恒指收益率 | +0.49% |
| 恒指起点 | 23006 |

### 6.3 悲剧根源分析

**问题**：工商银行2025年3月准确率90.48%，但平均收益率-4.53%

**根本原因**：
1. **股票特定因素**：工商银行的市场关联性仅0.05，几乎不受市场环境影响
2. **非市场因素驱动**：业绩不及预期、负面新闻、行业政策等股票特定因素
3. **模型局限性**：模型基于市场环境特征训练，对股票特定因素敏感度不足

**模型表现评估**：
- ✅ 准确率90.48%：模型正确预测了市场环境（震荡市）
- ❌ 收益率-4.53%：无法预测股票特定风险

### 6.4 解决方案建议

1. **增加股票特定风险因子**
   - 财务指标偏离度
   - 新闻情感变化率
   - 成交量异常检测
   - 行业板块轮动指标

2. **动态风险控制**
   - 在震荡市环境下降低仓位
   - 对低置信度预测增加权重
   - 设置止损点控制风险

3. **组合管理**
   - 分散投资，降低单一股票风险
   - 结合高市场关联性和低市场关联性股票
   - 动态调整配置比例

**市场关联性公式**：
```
市场关联性 = (牛市收益率 - 熊市收益率) / |牛市收益率 + 熊市收益率|
```

**正确决策比例公式**：
```
正确决策比例 = 胜率 × 准确率 / 100
```

**市场环境判定标准**：
```
牛市：月收益率 > +3%
熊市：月收益率 < -3%
震荡市：月收益率在 -3% 至 +3% 之间
```

### 附录C：术语说明

| 术语 | 说明 |
|------|------|
| 市场关联性 | 股票收益率与市场环境的相关程度，反映股票受市场影响的程度 |
