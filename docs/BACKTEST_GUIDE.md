# 回测评估指南

> **最后更新**：2026-04-27

---

## 概述

回测评估用于验证机器学习模型在真实交易环境中的盈利能力。

**推荐方法**：使用 **Walk-forward 验证**（见 `docs/VALIDATION_GUIDE.md`），而非简单回测。

简单回测不重新训练模型，结果虚高，不推荐用于最终决策。

---

## 关键指标

### 收益指标

| 指标 | 说明 | 优秀阈值 |
|------|------|---------|
| 年化收益率 | 按年化计算的收益率 | > 30% |
| 夏普比率 | 风险调整后收益 | > 1.0 |
| 索提诺比率 | 下行风险调整后收益 | > 1.0 |

### 风险指标

| 指标 | 说明 | 优秀阈值 |
|------|------|---------|
| 最大回撤 | 历史最大亏损幅度 | < -20% |
| 波动率 | 收益率标准差 | 低 |

### 交易指标

| 指标 | 说明 | 优秀阈值 |
|------|------|---------|
| 胜率 | 盈利交易占比 | > 50% |
| 交易次数 | 完整买卖循环数 | 适中 |

---

## 交易策略

- 当预测概率 > 置信度阈值（默认0.55）时买入
- 当预测概率 ≤ 置信度阈值时卖出
- 交易成本：佣金0.1% + 印花税0.1% + 滑点0.1% ≈ 0.5%

---

## 使用方法

### Walk-forward 验证（推荐）

```bash
python3 ml_services/walk_forward_validation.py --model-type catboost --horizon 20
```

### 简单回测（仅供参考）

```bash
python3 ml_services/batch_backtest.py --model-type catboost --horizon 20
```

## 评价标准

| 夏普比率 | 最大回撤 | 评级 |
|---------|---------|------|
| > 1.0 | < -20% | ⭐⭐⭐⭐⭐ 优秀 |
| > 0.5 | < -30% | ⭐⭐⭐⭐ 良好 |
| > 0.0 | < -40% | ⭐⭐⭐ 一般 |
| ≤ 0.0 | ≥ -40% | ⭐⭐ 较差 |

---

## 注意事项

1. **推荐 Walk-forward**：简单回测结果虚高，Walk-forward 更可信
2. **CatBoost 1天模型**：存在过拟合风险，不推荐使用
3. **样本量要求**：建议至少 252 个交易日数据
4. **交易成本**：港股双边成本约 0.5%
