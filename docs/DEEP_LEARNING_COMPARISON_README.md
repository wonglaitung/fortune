# 深度学习模型对比实验

## 结论

**CatBoost 远优于深度学习模型**，推荐继续使用 CatBoost 作为主要预测模型。

## 实验结果

| 模型 | 平均准确率 | 平均F1分数 | 训练时间 | 推荐度 |
|------|-----------|-----------|----------|--------|
| **CatBoost** | **65.10%** | **0.6361** | 快（1-2分钟） | ⭐⭐⭐⭐⭐ |
| LSTM | 51.79% | **0.0000** | 慢（1-2分钟） | ⭐ |
| Transformer | 51.15% | 0.1303 | 慢（2-3分钟） | ⭐ |

## 关键发现

**LSTM 问题**：
- F1分数为 0：完全无法识别上涨信号
- 预测概率集中在 0.48-0.49（接近随机）
- 回测无交易触发

**Transformer 问题**：
- 准确率仅比随机略好
- F1分数很低（0.1303）
- 训练时间长

**CatBoost 优势**：
- 准确率领先（65% vs 51%）
- F1分数优秀（0.6361）
- 训练速度快（比深度学习快 5-10 倍）

## 推荐

- ✅ **使用 CatBoost 20天模型**
- ❌ **不推荐深度学习模型**（LSTM/Transformer）
