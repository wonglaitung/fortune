## 三、板块领先-滞后关系分析 ⏰

### 3.1 板块与恒指的同步性

**关键发现**: 所有板块与恒指**同步**（0天滞后），没有明显的领先或滞后关系。

这说明港股板块高度市场化，信息传递迅速，**没有明显的领先指标板块**。

### 3.2 板块间轮动顺序

虽然没有严格的领先滞后，但可以通过**动量切换**识别轮动：

```
轮动链条:

牛市启动:
科技股/消费股 先行 → 半导体/生物医药 跟进 → 能源/航运 补涨
   (贝塔最高)          (成长性最强)          (周期末端)

熊市防御:
公用事业 先行 → 银行/保险 跟进 → 整体减仓
   (最独立)          (防御性跟随)
```

## 六、板块轮动核心规律 💡

### 6.1 时间关系总结

| 关系类型 | 发现 | 应用 |
|---------|------|------|
| **轮动周期** | 约28天一个完整周期 | 每2-3周评估轮动时机 |
| **持续性** | 防御占优期(16.7天) > 周期占优期(11.8天) | 熊市更持久，牛市更短暂 |
| **同步性** | 所有板块与HSI同步 | 无需寻找领先指标 |

### 6.2 因果关系总结

| 因果链 | 解释 | 策略 |
|--------|------|------|
| **大盘→板块** | HSI决定板块方向 | 先看大盘，再选板块 |
| **周期↔防御** | 负相关轮动 | 跷跷板效应，此消彼长 |
| **内部传导** | 消费-科技-保险铁三角 | 一起启动，一起结束 |

### 6.3 轮动规律总结


## 七、实用轮动策略 📋

### 7.1 状态判断框架

```python
def judge_market_regime(hsi_return_20d):
    """判断市场状态"""
    if hsi_return_20d > percentile(hsi, 75%):
        return "BULL"  # 牛市
    elif hsi_return_20d < percentile(hsi, 25%):
        return "BEAR"  # 熊市
    else:
        return "NEUTRAL"  # 震荡

def select_sectors(regime):
    """根据状态选板块"""
    if regime == "BULL":
        return ["semiconductor", "biotech", "tech", "consumer"]  # 周期
    elif regime == "BEAR":
        return ["insurance", "utility", "bank"]  # 防御
    else:
        return dynamic_balance()  # 动态平衡
```

## 八、关键发现总结 🔑

### 8.1 时间关系
1. **轮动周期**: 28天一个完整周期（牛市12天+熊市17天）
2. **板块同步**: 所有板块与恒指同步，无领先指标
3. **持续性**: 熊市防御期比牛市周期期更长

### 8.2 因果关系
1. **大盘决定板块**: HSI方向决定板块轮动方向
2. **跷跷板效应**: 周期与防御板块负相关轮动
3. **铁三角联动**: 消费-科技-保险高度相关，同涨同跌

### 8.3 轮动规律
1. **领涨板块**: 半导体（19.0%）和生物医药（17.4%）领涨频率最高
2. **牛市策略**: 74.3%时间周期板块占优，重仓成长
3. **熊市策略**: 65.9%时间防御板块占优，切换防御
4. **动量效应**: 动量相关性普遍较低，不建议单独使用动量/反转策略

### 8.4 交易启示

| 策略 | 条件 | 操作 | 置信度 |
|------|------|------|--------|
| **周期配置** | 牛市（HSI 20天收益>75%分位） | 半导体、科技、消费 | ⭐⭐⭐⭐ |
| **防御配置** | 熊市（HSI 20天收益<25%分位） | 保险、公用事业、银行 | ⭐⭐⭐⭐ |
| **轮动** | HSI 20天收益转换 | 周期↔防御切换 | ⭐⭐⭐ |
| **避险** | 震荡市 | 公用事业+现金 | ⭐⭐⭐ |
| **动量/反转** | 单独使用 | ⚠️ 不推荐 | ⭐⭐ |
