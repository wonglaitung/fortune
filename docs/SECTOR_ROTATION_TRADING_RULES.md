**版本**: v1.1

## 一、核心交易法则

### 法则1: 大盘决定方向（最高优先级）

```python
# 每日开盘前执行
hsi_return_20d = 恒生指数过去20日收益率

if hsi_return_20d > 历史75%分位:
    market_regime = "BULL"  # 牛市
elif hsi_return_20d < 历史25%分位:
    market_regime = "BEAR"  # 熊市
else:
    market_regime = "NEUTRAL"  # 震荡
```

**执行标准**:
- 牛市: 满仓操作，重仓周期性板块
- 熊市: 减仓至75%，只持有防御性板块
- 震荡: 半仓操作，根据相对强弱动态调整

### 法则3: 轮动转换信号

#### 3.1 牛市→熊市转换

**触发条件**（满足2项以上）:
1. HSI 20天收益从正转负
2. 周期性板块相对收益从高点回落超过30%
3. 防御性板块开始领涨（连续3天跑赢大盘）
4. 成交量萎缩至20日均量以下

**操作流程**:
```
Day 1: 触发信号
   ↓
   立即减仓50%
   ↓
Day 2-3: 确认信号
   ↓
   完成调仓至熊市配置
   ↓
后续: 持有防御性板块
```

#### 3.2 熊市→牛市转换

**触发条件**（满足2项以上）:
1. HSI 20天收益从负转正且持续改善
2. 半导体或生物医药板块领涨（连续5天跑赢大盘）
3. 成交量放大至20日均量以上20%
4. CatBoost预测HSI上涨置信度>0.6

**操作流程**:
```
Day 1: 触发信号
   ↓
   建仓周期性板块30%
   ↓
Day 2-5: 确认信号
   ↓
   完成调仓至牛市配置
   ↓
后续: 持有周期性板块
```

### 法则4: 板块内个股选择

#### 4.1 选股条件（必须全部满足）

1. **CatBoost预测**: `prediction == 1` (上涨)
2. **置信度**: `probability > 0.6`
3. **趋势确认**: 股价高于20日均线
4. **成交量**: 当日成交量 > 20日均量

#### 4.2 选股优先级

```python
def stock_priority(score, probability, sector_rank):
    """
    个股优先级评分
    """
    return (score * 0.4 + 
            probability * 0.3 + 
            (1 / sector_rank) * 0.3)

# 优先选择优先级评分最高的3只股票
```

### 法则5: 仓位管理

#### 5.1 基础仓位

| 市场状态 | 总仓位 | 单板块最大 | 单股最大 |
|---------|--------|-----------|---------|
| 牛市 | 90% | 30% | 15% |
| 震荡 | 60% | 20% | 10% |
| 熊市 | 75% | 30% | 15% |

#### 5.2 动态调整

**加仓条件**（满足后可增加10%仓位）:
- 当前配置持仓满5天且收益为正
- CatBoost预测继续看涨
- 无新的风险信号

**减仓条件**（必须减仓10%）:
- 单板块亏损超过5%
- 市场状态转换信号出现
- 连续3天跑输大盘

---

### 法则6: 风险控制

#### 6.1 止损规则

| 止损类型 | 触发条件 | 操作 |
|---------|---------|------|
| **个股止损** | 亏损 > 7%（牛市）/ 6%（熊市） | 立即卖出 |
| **板块止损** | 板块指数跌破20日均线 | 减仓50% |
| **大盘止损** | HSI跌破60日均线 | 整体仓位降至30% |
| **轮动失败** | 轮动后2周收益为负 | 回归原配置 |

#### 6.2 止盈规则

| 止盈类型 | 触发条件 | 操作 |
|---------|---------|------|
| **动态止盈** | 收益>15%后回撤5% | 卖出50% |
| **目标止盈** | 20天持有期满+收益>10% | 卖出50%，持有剩余 |
| **轮动止盈** | 市场状态转换信号出现 | 按转换流程操作 |

#### 6.3 最大回撤控制

- 单日最大亏损不超过总资产的3%
- 单月最大亏损不超过总资产的10%
- 触及最大回撤后，强制减仓至50%

### 法则7: 交易执行清单

#### 7.1 每日开盘前检查清单

```
□ 1. 计算HSI 20天收益，判断市场状态
□ 2. 检查轮动转换信号
□ 3. 检查持仓个股CatBoost预测
□ 4. 检查止损条件
□ 5. 检查板块相对强弱排名
□ 6. 制定当日交易计划
```

#### 7.2 每周五收盘后检查清单

```
□ 1. 重新评估所有板块Walk-forward表现
□ 2. 更新板块轮动统计
□ 3. 检查CatBoost模型预测准确率
□ 4. 评估当前配置是否需要调整
□ 5. 制定下周交易计划
```

## 二、具体交易场景

### 场景1: 牛市启动初期

**识别信号**:
- HSI 20天收益从负转正
- 半导体板块领涨3天

**操作步骤**:
1. 立即建仓半导体30%（0981.HK, 1347.HK）
2. 建仓生物医药20%（2269.HK）
3. 保留50%现金等待确认

**确认加仓**（3天后）:
- 如果收益为正: 加仓至牛市满仓配置
- 如果收益为负: 减仓至10%，等待更明确信号

### 场景2: 熊市反弹陷阱

**识别信号**:
- HSI短期反弹但20天收益仍为负
- 周期性板块反弹但防御性板块未跟随

**操作步骤**:
1. 不参与反弹
2. 保持熊市配置
3. 等待HSI 20天收益转正

**原因**: 熊市中周期/防御比例仅0.52，追涨周期性板块风险高

### 场景3: 震荡市板块轮动

**识别信号**:
- HSI 20天收益在历史25%-75%之间
- 板块轮动加速

**操作步骤**:
1. 每5天评估板块20日相对强弱
2. 持有相对强弱前3的板块
3. 单个板块不超过20%
4. 保留20%现金

### 场景4: 防御性板块配置

**识别信号**:
- HSI 20天收益低于25%分位
- 市场进入熊市状态

**操作步骤**:
1. 优先配置保险股（30%）和公用事业（25%）
2. 银行股作为辅助（20%），需等待CatBoost看涨信号
3. 保留25%现金
4. 持有期至少20天

**理论依据**: 熊市中防御板块占优比例65.9%，但银行股Walk-forward验证表现一般（夏普-0.053），不宜重仓

## 三、交易纪律

### 3.1 必须遵守的铁律

1. **大盘优先**: 任何交易必须先判断市场状态
2. **板块其次**: 个股选择必须在正确的板块内进行
3. **止损铁律**: 触及止损必须立即执行，不得犹豫
4. **轮动确认**: 板块轮动必须有至少2个信号确认
5. **仓位控制**: 单股不超过15%，单板块不超过30%

### 3.2 禁止行为

❌ **禁止行为**:
- 在没有CatBoost预测的情况下买入
- 在熊市中重仓周期性板块
- 止损后没有新信号就重新买入同一股票
- 单只股票仓位超过15%
- 追涨已经上涨超过10%的股票

### 3.3 允许例外

✅ **允许例外**（需记录理由）:
- 重大利好消息导致的技术性突破
- 市场极端情绪导致的超跌反弹机会
- 模型预测准确率连续低于50%时的手工调整

## 四、策略评估与优化

### 4.1 关键绩效指标(KPI)

| 指标 | 目标 | 评估频率 |
|------|------|---------|
| 年化收益率 | >15% | 每月 |
| 最大回撤 | <-20% | 每月 |
| 胜率 | >55% | 每月 |
| 夏普比率 | >1.0 | 每季度 |
| 轮动准确率 | >70% | 每季度 |

### 4.2 策略失效信号

| 失效信号 | 阈值 | 应对措施 |
|---------|------|---------|
| 连续亏损月份 | 3个月 | 减仓至30%，重新验证模型 |
| 轮动准确率 | <50% | 暂停轮动，持有现金 |
| 最大回撤 | >25% | 强制止损，重新评估策略 |
| CatBoost准确率 | <52% | 停止使用模型，等待重新训练 |

### 4.3 定期回测

- **频率**: 每季度进行一次完整回测
- **数据**: 过去2年数据
- **验证**: Walk-forward 12 folds
- **调整**: 根据回测结果调整权重和阈值
