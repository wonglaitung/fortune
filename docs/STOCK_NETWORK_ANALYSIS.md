## 一、股票网络分析概述

### 1.1 基本概念

股票网络分析将每只股票视为一个**节点**，股票之间的相关性视为**边**，构建网络图：

```
股票A ←──相关性0.85──→ 股票B
  │
  └──相关性0.72──→ 股票C
```

### 1.2 七种网络方法

| 方法 | 数据类型 | 特点 | 主要用途 |
|------|----------|------|----------|
| **MST** | 收益率 | 只保留最重要的连接 | 找核心枢纽 |
| **PMFG** | 收益率 | 保留更多连接 | 社区检测 |
| **阈值网络** | 收益率 | 只保留相关性 > 阈值 | 强关联分析 |
| **偏相关网络** | 收益率残差 | 剔除市场因子影响 | 因果关系推断 |
| **成交量网络** | 成交量变化率 | 反映流动性联动 | 流动性风险 |
| **动量网络** | N日收益率 | 反映趋势同步 | 趋势策略 |
| **波动率网络** | 滚动波动率 | 反映风险传导 | 系统性风险 |

### 1.3 网络拓扑指标对比

| 指标 | MST | 阈值网络 | 成交量网络 | 动量网络 | 波动率网络 | 含义 |
|------|-----|----------|-----------|---------|-----------|------|
| 边数 | 45 | 153 | 9 | 180 | 362 | 连接数量 |
| 密度 | 0.04 | 0.15 | 0.01 | 0.17 | 0.35 | 连接紧密程度 |
| 聚类系数 | 0.00 | 0.53 | 0.07 | 0.61 | 0.65 | 节点抱团程度 |
| 平均路径 | 3.60 | 1.85 | - | - | - | 信息传递距离 |

**关键发现**：
- **波动率网络密度最高（0.35）**：风险传导性最强
- **动量网络聚类系数高（0.61）**：趋势同步性强
- **成交量网络密度最低（0.01）**：流动性联动弱，资金流向相对独立

## 二、网络方法详解

### 2.1 MST（最小生成树）

**原理**：从完全图中选择 N-1 条最重要的边，使得所有节点连通且总距离最小。

**距离定义**：Mantegna 距离 = √(2 × (1 - 相关系数))

```
完全图（所有股票两两相连）
    ↓ MST算法
树状图（只保留最重要的 N-1 条边）
```

**特点**：
- 形成树状结构，无环路
- 边数 = 节点数 - 1
- 清晰展示核心枢纽

**用途**：识别系统性重要股票、构建层级结构

### 2.2 PMFG（平面最大过滤图）

**原理**：在 MST 基础上保留更多连接，形成平面图（可以在平面上绘制且边不相交）。

**特点**：
- 边数 ≈ 3 × (N - 2)
- 保留三角形结构，适合社区检测
- 比 MST 保留更多信息

**用途**：社区检测、分析股票群落

### 2.3 阈值网络

**原理**：只保留相关系数绝对值 > 阈值（默认0.5）的边。

```
股票A vs 股票B: 相关系数 = 0.85 > 0.5 → 保留边
股票A vs 股票C: 相关系数 = 0.35 < 0.5 → 不保留边
```

**特点**：
- 简单直观
- 可调节阈值控制网络稀疏程度
- 可能产生孤立节点

**用途**：强关联分析、识别高相关股票组合

### 2.4 偏相关网络

**原理**：剔除市场因子（所有股票平均收益）影响后，计算残差相关性。

```python
# 市场因子
market = 所有股票平均收益率

# 对每只股票回归
收益率 = α + β × 市场因子 + 残差

# 用残差计算偏相关
偏相关 = 残差的相关系数
```

**特点**：
- 剔除大盘影响，揭示股票间"真实"关联
- 边数通常较少（更严格的关联）

**用途**：因果关系推断、发现隐藏关联

### 2.5 成交量网络

**原理**：计算成交量变化率的相关性。

```python
# 成交量变化率
volume_change = Volume.pct_change()

# 成交量相关系数
volume_corr = volume_change.corr()
```

**特点**：
- 反映资金流动同步性
- 通常密度较低（资金流向独立）

**用途**：流动性风险管理、资金流向追踪

### 2.6 动量网络

**原理**：计算 N 日收益率（动量）的相关性。

```python
# 20日动量
momentum_20d = Close.pct_change(20)

# 动量相关系数
momentum_corr = momentum_20d.corr()
```

**特点**：
- 反映趋势同步性
- 聚类系数通常较高（趋势抱团）

**用途**：趋势跟踪策略、板块轮动分析

### 2.7 波动率网络

**原理**：计算滚动波动率的相关性。

```python
# 20日滚动波动率
volatility_20d = Return.rolling(20).std()

# 波动率相关系数
vol_corr = volatility_20d.corr()
```

**特点**：
- 反映风险传导性
- 密度通常最高（波动率传导强）

**用途**：系统性风险预警、风险对冲

## 四、社区检测分析

### 4.1 社区检测概念

社区检测将股票分成若干"群落"，同一群落内的股票相关性更高：

```
社区0: 汇丰银行、腾讯、阿里、百度... (科技+银行混合)
社区1: 中国平安、中国太保、中国人寿... (保险股)
社区2: 建设银行、农业银行、工商银行... (银行股)
社区3: 美团、小米、京东、中芯国际... (科技股)
社区4: 中电控股、电能实业、友邦保险... (公用事业+保险)
```

### 11.4 网络特征的正确使用方式

#### 为什么网络特征不适合放入个股预测模型？

| 原因 | 说明 |
|------|------|
| **模型假设冲突** | 个股模型假设 IID，但网络特征本质是描述"不独立性"，在市场极端波动时会导致预测偏误 |
| **维度灾难** | 网络特征呈长尾分布，极端值（权重股/妖股）会导致过拟合 |
| **信噪比低** | 网络拓扑每天变化 <0.1%，每日更新带来的只是计算开销，而非预测增量 |

#### 数据泄漏防护（已实施）

**问题识别**（2026-05-02）：
- `build_returns_dataframe()` 直接使用当日收益率数据
- `calculate_node_deviation()` 包含当日收益率

**修复措施**：
1. **收益率数据时间偏移**：
   ```python
   # 修复前（存在泄漏）
   returns_dict[stock_code] = df['Return']

   # 修复后（安全）
   returns_dict[stock_code] = df['Return'].shift(1)  # 排除当日
   ```

2. **节点偏离度计算**：
   ```python
   # 修复前（存在泄漏）
   scores = returns_df.iloc[-window:].sum()  # 包含当日

   # 修复后（安全）
   scores = returns_df.iloc[-window-1:-1].sum()  # 排除当日
   ```

3. **统一使用 `for_prediction=True` 参数**：
   - 所有网络特征计算默认排除当日数据
   - 确保使用 $T-1$ 日数据构建网络，预测 $T$ 日

**验证方法**：
- 检查 `build_returns_dataframe(stock_data, for_prediction=True)` 调用
- 确保所有特征计算不包含 `returns_df.iloc[-1]`（当日数据）

#### 正确用法

| 用法 | 说明 |
|------|------|
| **风险对冲** | 波动率网络密度 > 阈值 → 市场进入"同涨同跌"模式 → 降低仓位 |
| **逻辑触发更新** | 监控邻接矩阵 Frobenius 范数变化率，超过阈值才重新计算 |
| **残差化处理** | 网络特征对动量/成交量因子回归，只取残差作为输入 |

#### 波动率网络密度预警原理

**密度定义**：
```
密度 = 实际边数 / 最大可能边数
     = 股票间波动高度相关的比例
```

**预警阈值**（基于过去30天滚动计算）：

| 密度水平 | 含义 | 市场状态 | 操作建议 |
|---------|------|----------|----------|
| **低（<0.3）** | 波动独立 | 个股分化，Alpha 有效 | 正常交易 |
| **中（0.3-0.45）** | 波动联动 | 板块轮动，需关注 | 谨慎交易 |
| **高（>0.45）** | 波动高度同步 | "同涨同跌"模式，个股模型失效 | **降低仓位** |

**核心逻辑**：
```
密度高 → 市场进入"同涨同跌"模式
       → 个股分化度低
       → 选股模型（寻找 Alpha）失效
       → 应降低仓位，而非寻找个股机会
```

**与传统风险指标的区别**：

| 指标 | VIX | 波动率网络密度 |
|------|-----|---------------|
| 测量对象 | 市场整体恐慌程度 | 股票间风险传导强度 |
| 预警类型 | 市场下跌风险 | **选股失效风险** |
| 适用场景 | 择时 | **仓位管理** |
