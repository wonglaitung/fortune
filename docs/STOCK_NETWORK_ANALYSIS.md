## 一、什么是股票网络分析？

### 1.1 基本概念

股票网络分析将每只股票视为一个**节点**，股票之间的相关性视为**边**，构建一个网络图。

```
示例：
股票A ←──相关性0.85──→ 股票B
  │
  └──相关性0.72──→ 股票C
```

### 1.2 七种网络方法对比

| 方法 | 全称 | 数据类型 | 特点 | 用途 |
|------|------|----------|------|------|
| **MST** | 最小生成树 | 收益率 | 只保留最重要的连接 | 找核心枢纽 |
| **PMFG** | 平面最大过滤图 | 收益率 | 保留更多连接 | 社区检测 |
| **阈值网络** | 阈值网络 | 收益率 | 只保留相关性 > 阈值 | 强关联分析 |
| **偏相关网络** | 偏相关网络 | 收益率残差 | 剔除市场因子影响 | 因果关系推断 |
| **成交量网络** | 成交量相关性网络 | 成交量变化率 | 反映流动性联动 | 流动性风险 |
| **动量网络** | 动量相关性网络 | N日收益率 | 反映趋势同步 | 趋势策略 |
| **波动率网络** | 波动率相关性网络 | 滚动波动率 | 反映风险传导 | 系统性风险 |

#### 1.2.1 MST（最小生成树）

**原理**：从完全图中选择 N-1 条最重要的边，使得所有节点连通且总距离最小。

**距离定义**：Mantegna 距离 = √(2 × (1 - 相关系数))

```
完全图（所有股票两两相连）
    ↓ MST算法
树状图（只保留最重要的 N-1 条边）
```

**特点**：
- 形成树状结构，无环路
- 边数 = 节点数 - 1
- 清晰展示核心枢纽

**用途**：识别系统性重要股票、构建层级结构

#### 1.2.2 PMFG（平面最大过滤图）

**原理**：在 MST 基础上保留更多连接，形成平面图（可以在平面上绘制且边不相交）。

**特点**：
- 边数 ≈ 3 × (N - 2)
- 保留三角形结构，适合社区检测
- 比 MST 保留更多信息

**用途**：社区检测、分析股票群落

#### 1.2.3 阈值网络

**原理**：只保留相关系数绝对值 > 阈值（默认0.5）的边。

```
股票A vs 股票B: 相关系数 = 0.85 > 0.5 → 保留边
股票A vs 股票C: 相关系数 = 0.35 < 0.5 → 不保留边
```

**特点**：
- 简单直观
- 可调节阈值控制网络稀疏程度
- 可能产生孤立节点

**用途**：强关联分析、识别高相关股票组合

#### 1.2.4 偏相关网络

**原理**：剔除市场因子（所有股票平均收益）影响后，计算残差相关性。

```python
# 市场因子
market = 所有股票平均收益率

# 对每只股票回归
收益率 = α + β × 市场因子 + 残差

# 用残差计算偏相关
偏相关 = 残差的相关系数
```

**特点**：
- 剔除大盘影响，揭示股票间"真实"关联
- 边数通常较少（更严格的关联）

**用途**：因果关系推断、发现隐藏关联

#### 1.2.5 成交量网络

**原理**：计算成交量变化率的相关性。

```python
# 成交量变化率
volume_change = Volume.pct_change()

# 成交量相关系数
volume_corr = volume_change.corr()
```

**特点**：
- 反映资金流动同步性
- 通常密度较低（资金流向独立）

**用途**：流动性风险管理、资金流向追踪

#### 1.2.6 动量网络

**原理**：计算 N 日收益率（动量）的相关性。

```python
# 20日动量
momentum_20d = Close.pct_change(20)

# 动量相关系数
momentum_corr = momentum_20d.corr()
```

**特点**：
- 反映趋势同步性
- 聚类系数通常较高（趋势抱团）

**用途**：趋势跟踪策略、板块轮动分析

#### 1.2.7 波动率网络

**原理**：计算滚动波动率的相关性。

```python
# 20日滚动波动率
volatility_20d = Return.rolling(20).std()

# 波动率相关系数
vol_corr = volatility_20d.corr()
```

**特点**：
- 反映风险传导性
- 密度通常最高（波动率传导强）

**用途**：系统性风险预警、风险对冲

### 1.3 网络拓扑指标解读

| 指标 | MST | 阈值网络 | 成交量网络 | 动量网络 | 波动率网络 | 含义 |
|------|-----|----------|-----------|---------|-----------|------|
| **边数** | 45 | 153 | 9 | 180 | 362 | 连接数量 |
| **密度** | 0.04 | 0.15 | 0.01 | 0.17 | 0.35 | 连接紧密程度 |
| **聚类系数** | 0.00 | 0.53 | 0.07 | 0.61 | 0.65 | 节点抱团程度 |
| **平均路径** | 3.60 | 1.85 | - | - | - | 信息传递距离 |

**关键发现**：
- **波动率网络密度最高（0.35）**：风险传导性最强
- **动量网络聚类系数高（0.61）**：趋势同步性强
- **成交量网络密度最低（0.01）**：流动性联动弱，资金流向相对独立

## 二、系统性重要股票（网络中心性）

### 2.1 什么是中心性？

中心性衡量一只股票在网络中的"重要性"：

| 中心性类型 | 含义 | 高中心性股票特点 |
|-----------|------|------------------|
| **度中心性** | 直接连接数 | 与很多股票相关联 |
| **介数中心性** | 经过该节点的最短路径数 | 信息传递必经之路 |
| **特征向量中心性** | 连接节点的重要性 | 连接重要股票的股票 |
| **接近中心性** | 到其他节点的平均距离 | 能快速影响其他股票 |

## 三、社区检测分析

### 3.1 什么是社区检测？

社区检测将股票分成若干"群落"，同一群落内的股票相关性更高：

```
社区0: 汇丰银行、腾讯、阿里、百度... (科技+银行混合)
社区1: 中国平安、中国太保、中国人寿... (保险股)
社区2: 建设银行、农业银行、工商银行... (银行股)
社区3: 美团、小米、京东、中芯国际... (科技股)
社区4: 中电控股、电能实业、友邦保险... (公用事业+保险)
```

| 社区ID | 股票数 | 主要板块 | 代表股票 |
|--------|--------|----------|----------|

**板块错位** 指股票的网络行为与其官方板块不一致：

## 七、多维度网络分析（新增）

### 7.1 成交量相关性网络

**数据**：成交量变化率（`Volume.pct_change()`）

**目的**：识别流动性联动的股票组合

#### 7.1.1 网络拓扑指标

| 指标 | 数值 | 解读 |
|------|------|------|
| 边数 | 9 | 只有9对股票成交量高度相关 |
| 密度 | 0.01 | 流动性联动非常弱 |
| 聚类系数 | 0.07 | 几乎不抱团 |

#### 7.1.2 成交量网络核心枢纽

由于成交量网络边数很少（仅9条），说明港股成交量联动非常弱。这意味着：

- **资金流向独立**：不同股票的资金流入/流出相对独立
- **流动性风险分散**：单一股票流动性问题不太会传导到其他股票

#### 7.1.3 成交量高相关组合

成交量相关性 > 0.5 的股票组合（仅9对）：

| 组合 | 相关性 | 解读 |
|------|--------|------|
| 银行股之间 | 较高 | 同板块资金流向同步 |
| 科技股之间 | 较高 | 同板块资金流向同步 |

**发现**：港股成交量联动弱，资金流向相对独立。

**应用场景**：
- **流动性风险管理**：当市场流动性枯竭时，哪些股票会同时受影响
- **资金流向追踪**：发现资金同时流入/流出的股票群

### 7.2 动量相关性网络

**数据**：20日收益率（`Close.pct_change(20)`）

**目的**：识别趋势同步的股票组合

#### 7.2.1 网络拓扑指标

| 指标 | 数值 | 解读 |

#### 7.2.2 动量网络核心枢纽

动量网络聚类系数高达 **0.61**，说明股票趋势高度同步。这意味着：

- **趋势共振**：当市场趋势形成时，大部分股票会跟随
- **板块轮动明显**：同一板块的股票趋势高度一致

#### 7.2.3 动量高相关组合

动量相关性 > 0.5 的股票组合（180对）：

| 类型 | 数量 | 解读 |
|------|------|------|
| 同板块高相关 | 较多 | 板块内趋势同步 |
| 跨板块高相关 | 较多 | 市场整体趋势驱动 |

**发现**：动量网络聚类系数高达0.61，说明股票趋势高度同步。

**应用场景**：
- **趋势跟踪策略**：发现趋势同步的股票，确认趋势信号
- **板块轮动**：识别动量轮动模式

**目的**：识别风险传导的股票组合

|------|------|------|

波动率网络密度最高（**0.35**），聚类系数最高（**0.65**），说明：

- **风险传导强**：当一只股票波动加剧时，会传导到其他股票
- **系统性风险高**：市场波动会同时影响大部分股票

#### 7.3.3 波动率高相关组合

波动率相关性 > 0.5 的股票组合（362对）：

| 类型 | 数量 | 解读 |
|------|------|------|
| 全市场高相关 | 362对 | 波动率传导覆盖全市场 |
| 板块内高相关 | 较多 | 板块内波动同步 |

**发现**：波动率网络密度最高（0.35），风险传导性最强。

**应用场景**：
- **系统性风险预警**：当波动率网络密度上升时，警惕系统性风险
- **风险对冲**：选择波动率相关性低的股票进行对冲

### 7.4 四种网络对比

| 网络类型 | 边数 | 密度 | 聚类系数 | 核心发现 | 应用场景 |
|---------|------|------|----------|----------|----------|
| 价格相关性 | 153 | 0.15 | 0.53 | 盈富基金为核心枢纽 | 系统性风险 |
| 成交量相关性 | **9** | **0.01** | 0.07 | 流动性联动弱 | 流动性风险 |
| 动量相关性 | 180 | 0.17 | **0.61** | 趋势高度同步 | 趋势策略 |
| 波动率相关性 | **362** | **0.35** | **0.65** | 风险传导最强 | 系统性风险 |

**关键发现**：
1. **波动率网络最密集**：风险传导性最强，需警惕系统性风险
2. **动量网络聚类最高**：趋势同步性强，适合趋势跟踪
3. **成交量网络最稀疏**：流动性联动弱，资金流向独立

### 7.5 多层网络分析

## 十、实际应用建议

### 10.1 风险管理

1. **监控核心枢纽**：盈富基金、恒生中国企业大跌时，警惕系统性风险
2. **关注桥梁股票**：盈富基金连接多个社区，其波动会跨板块传导
3. **避免集中持仓**：不要同时持有同一社区的多只股票
4. **波动率网络监控**：当波动率网络密度上升时，警惕系统性风险

### 10.2 交易策略

1. **领先滞后策略**：
   - 绿色动力环保大涨 → 4天后关注药明生物、中国海洋石油
   - 汇丰银行大涨 → 2天后关注友邦保险

   - 注意板块错位股票（汇丰与科技股联动）
   - 不要按官方板块简单轮动

   - 使用推荐的5只股票组合

   - 当多只股票趋势一致时，确认趋势信号
