## 三、因果关系分析 🔗

### 3.1 预测正确性的传导效应

**最新验证结果**（2026-04-28，906个样本）：

| 条件 | 准确率 | 说明 |
|------|--------|------|
| **1天预测正确时 → 20天也正确** | **83.56%** | 传导有效 |
| **1天预测错误时 → 20天正确** | **78.95%** | 基准水平 |
| **5天预测正确时 → 20天也正确** | **81.95%** | 传导有效 |
| **1天+5天都正确时 → 20天也正确** | **81.49%** | 双重确认 |

**核心发现**：
- 1天预测正确时，20天准确率提升 4.6%（83.56% vs 78.95%）
- 5天预测正确时，20天准确率略高于基准（81.95% vs 78.95%）
- **传导效应存在但较弱**：短期正确对长期准确率有正向影响

**指标含义说明**：
- **"1天预测正确时 → 20天也正确"**：当1天预测方向正确时，20天预测也正确的概率
- **"1天+5天都正确时 → 20天也正确"**：当1天和5天预测方向都正确时，20天预测也正确的概率
- 这是**预测正确率的传导**，不是预测概率与实际方向的相关性

### 3.2 预测概率与实际方向的相关性

| 相关性 | 数值 | 解读 |
|--------|------|------|
| 1天预测概率 与 5天实际方向 r | **+0.22** | 正相关 |
| 5天预测概率 与 20天实际方向 r | **+0.35** | 正相关（较强） |
| 1天预测概率 与 20天实际方向 r | **+0.11** | 正相关 |

**指标含义说明**：
- **"5天预测概率 与 20天实际方向 相关性 r"**：5天模型预测上涨的概率值（0-1）与20天实际涨跌方向（涨=1，跌=0）的相关系数
- **r > 0（正向）**：预测概率越高，实际上涨概率也越高 → 顺势操作有效
- **恒指 r = +0.35**：预测概率与实际方向正相关，顺势操作有效

### 11.1 验证结果汇总

**有效验证股票数：12只**

| 指标 | 数值 | 对比恒指 |
|------|------|---------|
| 平均1d→5d准确率 | 51.74% | 64.14%（恒指） |

**关键发现**：

- 个股噪音更大，短期预测接近随机
- **预测概率与实际方向相关性为负值**：个股预测概率高反而预示反转，与恒指完全相反

**指标含义说明**：
- **"传导r"**：短期预测正确时，长期预测也正确的相关性（预测正确率的传导）
