## 二、时间关系分析 ⏰

### 2.1 持有期结构

```
时间轴: T(预测日) ---- T+1 ---- T+5 ---- T+20
         │          │        │        │
1天周期:  [━━━━━━━━] 持有期[T, T+1]
```

**关键发现**:
- 1天持有期完全包含在5天周期内（前20%时段）
- 1天持有期完全包含在20天周期内（前5%时段）
- 5天持有期完全包含在20天周期内（前25%时段）

### 2.2 时间层级关系

| 关系 | 描述 | 影响 |
|------|------|------|
| **短→长包含** | 短周期是长周期的"快照" | 1天表现影响5天，5天影响20天 |
| **累积效应** | 长周期是短周期的累积 | 20天收益 = 4个5天 ≈ 20个1天 |
| **噪音过滤** | 周期越长，噪音越少 | 1天噪音最大，20天趋势最清晰 |

## 三、因果关系分析 🔗

### 3.1 预测正确性的传导效应

**最新验证结果**（2026-04-28）：

| 条件 | 旧值（4/26） | 新值（4/28） | 变化 |
|------|---------|---------|------|
| 1天预测正确时 → 20天 | 80.94% | **78.91%** | -2.03% |
| 1天预测错误时 → 20天 | 83.60% | **81.61%** | -1.99% |
| 1+5天都正确时 → 20天 | 81.46% | **79.47%** | -1.99% |

**核心发现**：1天预测错误时，20天准确率仍然更高（81.61% > 78.91%）

**因果解释**:
1. 短期噪音大，1天预测错误可能是"反向指标"
2. 模型在短期方向判断失误时，长期趋势反而更清晰
3. 5天作为"桥梁"作用依然重要

### 3.2 概率值的相关性结构

| 相关性 | 旧值 | 新值 | 解读 |
|--------|------|------|------|
| 1天 vs 5天 | 0.288 | **0.211** | 正相关减弱 |
| 5天 vs 20天 | 0.258 | **0.354** | 正相关增强 |
| 1天 vs 20天 | 0.021 | **0.021** | 几乎无关 |

**因果洞察**:
```
1天预测 ←─弱相关─→ 5天预测 ←─强相关─→ 20天预测
     ↑_____________无关_____________↑
```

- **5天是关键桥梁**: 与20天相关性从0.258增强到0.354
- **直接传导弱**: 1天噪音太大，难以直接影响20天判断
- **层级过滤**: 5天过滤了1天的噪音，提炼出有效信号传递给20天

**因果解释**:
```
T日    T+1日  T+5日  T+20日
 │      │      │      │
 ├─短线假突破─┤      │
        │           │
        └─→ 真趋势向上 ←─┤
                      │

```

- 短期(1天): 假突破信号，模型识别为涨

- 长期(20天): 真实趋势向上
- 实际: 短线假突破后，长期趋势确立

**交易启示**:
> 当出现"短线看涨但中线看跌"信号时，如果长周期看涨，**相信长周期**，这是"假突破抄底"信号！

## 七、恒指核心结论 💡

### 7.1 三周期关系的本质

```
┌─────────────────────────────────────────────────────────┐
│                    时间层级结构                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   1天周期 (微观)                                         │
│   ├── 作用: 捕捉即时动量、噪音大                         │

│   └── 对20天影响: 微弱                                  │
│                                                         │
│   ↓ 过滤噪音                                             │
│                                                         │
│   5天周期 (中观) ←── 关键桥梁                             │
│   ├── 作用: 确认短期趋势、平衡噪音与信号                  │
│   ├── 准确率: 58.28%                                    │
│   └── 对20天影响: 显著                                  │
│                                                         │
│   ↓ 提炼趋势                                             │
│                                                         │
│   20天周期 (宏观)                                        │
│   ├── 作用: 捕捉长期趋势、噪音最小                       │

│   └── 结论: 最可靠的预测                                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 7.2 因果关系的四大规律

| 规律 | 描述 | 应用 |
|------|------|------|
| **传导律** | 短→中→长，正确性依次传导 | 5天对了要行动，20天更可靠 |
| **过滤律** | 周期越长，噪音越少 | 20天预测最值得信赖 |
| **一致律** | 三周期一致时准确率较高 | 一致看涨81.4%，一致看跌83.6% |
| **背离律** | 短涨长跌或短跌长涨需谨慎 | 假突破(101)是抄底信号 |
