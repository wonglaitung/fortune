```

         │          │        │        │
```

**关键发现**:

- 1天持有期完全包含在20天周期内（前5%时段）

### 2.2 时间层级关系

| 关系 | 描述 | 影响 |
|------|------|------|
| **短→长包含** | 短周期是长周期的"快照" | 1天表现影响5天，5天影响20天 |
| **累积效应** | 长周期是短周期的累积 | 20天收益 = 4个5天 ≈ 20个1天 |
| **噪音过滤** | 周期越长，噪音越少 | 1天噪音最大，20天趋势最清晰 |

## 三、因果关系分析 🔗

### 3.1 预测正确性的传导效应

**新发现**: 传导效应发生变化

| 条件 | 旧准确率 | 新准确率 | 变化 |
|------|---------|---------|------|
| 1天预测正确时 → 20天 | 82.24% | **80.94%** | -1.30% |
| 1天预测错误时 → 20天 | 79.14% | **83.60%** | +4.46% |
| 5天预测正确时 → 20天 | 82.23% | **83.88%** | +1.65% |
| 1+5天都正确时 → 20天 | 84.12% | **81.46%** | -2.66% |

**意外发现**：1天预测错误时，20天准确率反而更高（83.60% > 80.94%）

**因果解释**:
1. 短期噪音大，1天预测错误可能是"反向指标"
2. 模型在短期方向判断失误时，长期趋势反而更清晰
3. 5天作为"桥梁"作用更强（5天正确→20天准确率83.88%）

### 3.2 概率值的相关性结构

| 相关性 | 旧值 | 新值 | 解读 |
|--------|------|------|------|
| 1天 vs 5天 | 0.288 | **0.211** | 正相关减弱 |
| 5天 vs 20天 | 0.258 | **0.354** | 正相关增强 |
| 1天 vs 20天 | 0.021 | **0.021** | 几乎无关 |

**因果洞察**:
```
1天预测 ←─弱相关─→ 5天预测 ←─强相关─→ 20天预测
     ↑_____________无关_____________↑
```

- **5天是关键桥梁**: 与20天相关性从0.258增强到0.354
- **直接传导弱**: 1天噪音太大，难以直接影响20天判断
- **层级过滤**: 5天过滤了1天的噪音，提炼出有效信号传递给20天

### 4.2 新发现：假突破模式（1-0-1）成为最优

**现象**: 1天看涨、5天看跌、20天看涨，20天实际准确率**92.73%**

**因果解释**:
```
T日    T+1日  T+5日  T+20日
 │      │      │      │
 ├─短线假突破─┤      │
        │           │
        └─→ 真趋势向上 ←─┤
                      │
                   实际: 涨（92.73%准确）
```

- 短期(1天): 假突破信号，模型识别为涨
- 中期(5天): 修正判断，识别为跌
- 长期(20天): 真实趋势向上
- 实际: 短线假突破后，长期趋势确立

**交易启示**:
> 当出现"短线看涨但中线看跌"信号时，如果长周期看涨，**相信长周期**，这是"假突破抄底"信号！

### 4.3 震荡回调模式（1-1-0）准确率下降

**变化**: 从 90.15% 降至 78.57%

**原因分析**:
- 新增特征后，模型对"短涨长跌"的判断更加谨慎
- 日历效应和GARCH波动率可能影响了短期信号的权重
- 样本量较小（98个）导致波动较大

## 七、恒指核心结论 💡

### 7.1 三周期关系的本质

```
┌─────────────────────────────────────────────────────────┐
│                    时间层级结构                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   1天周期 (微观)                                         │
│   ├── 作用: 捕捉即时动量、噪音大                         │
│   ├── 准确率: 51.55%                                    │
│   └── 对20天影响: 微弱 (r=0.021)                        │
│                                                         │
│   ↓ 过滤噪音                                             │
│                                                         │
│   5天周期 (中观) ←── 关键桥梁                             │
│   ├── 作用: 确认短期趋势、平衡噪音与信号                  │
│   ├── 准确率: 64.35%                                    │
│   └── 对20天影响: 显著 (r=0.354)                        │
│                                                         │
│   ↓ 提炼趋势                                             │
│                                                         │
│   20天周期 (宏观)                                        │
│   ├── 作用: 捕捉长期趋势、噪音最小                       │
│   ├── 准确率: 82.23%                                    │
│   └── 结论: 最可靠的预测                                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 7.2 因果关系的四大规律

| 规律 | 描述 | 应用 |
|------|------|------|
| **传导律** | 短→中→长，正确性依次传导 | 5天对了要行动，20天更可靠 |
| **过滤律** | 周期越长，噪音越少 | 20天预测最值得信赖 |
| **一致律** | 三周期一致时准确率较高 | 一致看涨81.4%，一致看跌83.6% |
| **背离律** | 短涨长跌或短跌长涨需谨慎 | 假突破(101)是抄底信号 |
