## 三、因果关系分析 🔗

### 3.1 预测正确性的传导效应

**最新验证结果**（2026-05-14，905个样本）：

| 条件 | 准确率 | 说明 |
|------|--------|------|
| **1天预测正确时 → 20天也正确** | **81.17%** | 传导有效 |
| **1天预测错误时 → 20天正确** | **80.14%** | 基准水平 |
| **5天预测正确时 → 20天也正确** | **81.52%** | 传导有效 |
| **1天+5天都正确时 → 20天也正确** | **81.49%** | 双重确认 |

**核心发现**：
- 1天预测正确时，20天准确率与基准相近（81.56% vs 82.21%）
- 5天预测正确时，20天准确率略高于基准（84.33% vs 82.21%）
- **传导效应存在但较弱**：短期正确对长期准确率有正向影响

**指标含义说明**：
- **"1天预测正确时 → 20天也正确"**：当1天预测方向正确时，20天预测也正确的概率
- **"1天+5天都正确时 → 20天也正确"**：当1天和5天预测方向都正确时，20天预测也正确的概率
- 这是**预测正确率的传导**，不是预测概率与实际方向的相关性

### 3.2 预测概率与实际方向的相关性

| 相关性 | 数值 | 解读 |
|--------|------|------|
| 1天预测概率 与 5天实际方向 r | **+0.17** | 正相关 |
| 5天预测概率 与 20天实际方向 r | **+0.33** | 正相关（较强） |
| 1天预测概率 与 20天实际方向 r | **+0.12** | 正相关 |

**指标含义说明**：
- **"5天预测概率 与 20天实际方向 相关性 r"**：5天模型预测上涨的概率值（0-1）与20天实际涨跌方向（涨=1，跌=0）的相关系数
- **r > 0（正向）**：预测概率越高，实际上涨概率也越高 → 顺势操作有效
- **恒指 r = +0.35**：预测概率与实际方向正相关，顺势操作有效

```
T日    T+1日  T+5日  T+20日

        │           │
```

- 长期(20天): 真实趋势向上
- 实际: 短线假突破后，长期趋势确立

**交易启示**:
> 当出现"短线看涨但中线看跌"信号时，如果长周期看涨，**相信长周期**，这是"假突破抄底"信号！
