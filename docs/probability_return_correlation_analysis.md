# 预测概率与实际涨幅相关性分析

## 分析目的

验证CatBoost 20天模型的预测概率与实际涨幅是否存在正相关关系，为基于概率建立涨幅映射表提供理论依据。

## 数据来源

- **数据文件**: `output/backtest_20d_trades_20260308_043002.csv`
- **数据范围**: 2025年全年回测数据
- **样本数量**: 6,328条交易记录
- **分析日期**: 2026-03-14

## 分析方法

1. **相关性分析**: 计算预测概率与实际涨幅的Pearson相关系数
2. **分箱分析**: 将概率分为10个区间，统计每个区间的平均涨幅、胜率、标准差
3. **线性回归**: 建立概率→涨幅的线性回归模型，计算R²和P值

## 关键发现

### 1. 相关性分析

| 指标 | 数值 | 解释 |
|------|------|------|
| **相关系数** | **0.6289** | 强正相关 |
| **R²值** | **0.3956** | 概率能解释39.56%的涨幅变化 |
| **P值** | **<0.0001** | 极显著相关 |
| **回归方程** | 涨幅 = 0.3566 × 概率 - 0.1760 | |

**结论**: 预测概率与实际涨幅存在**强正相关关系**，理论成立。

## 理论验证

### 问题：预测的probability值越大，涨幅也越大？

**答案：是的！基于本项目6,328条回测数据的实证分析：**

1. **强正相关**：相关系数0.6289，属于统计学上的强正相关
2. **解释力强**：概率能解释约40%的涨幅变化（R²=0.3956）
3. **统计显著**：P值<0.0001，结果极显著
4. **实证支持**：从概率0.50到0.90，平均涨幅从-0.24%增长到11.73%
