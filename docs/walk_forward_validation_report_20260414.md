## 二、指标定义说明

### 2.1 核心指标定义

| 指标 | 定义 | 计算公式 | 说明 |
|------|------|---------|------|
| **胜率 (Win Rate)** | 盈利交易占比 | 盈利交易数 / 总交易数 | 只统计置信度 ≥ 0.65 的交易信号 |
| **准确率 (Accuracy)** | 预测正确率 | 预测正确数 / 总样本数 | 统计所有样本，无论置信度高低 |
| **夏普比率** | 风险调整收益 | 年化收益 / 年化标准差 | 衡量单位风险下的收益 |
| **索提诺比率** | 下行风险调整收益 | 年化收益 / 下行标准差 | 只考虑下行风险 |
| **最大回撤** | 最大损失幅度 | 峰值到谷底的最大跌幅 | 衡量风险控制能力 |

### 2.2 胜率与准确率的区别

```
胜率 (Win Rate):
  - 计算范围：仅统计置信度 ≥ 0.65 的交易信号
  - 含义：实际交易的盈利比例
  - 公式：盈利交易数 / 总交易数

准确率 (Accuracy):
  - 计算范围：统计所有样本
  - 含义：模型预测方向的正确率
  - 公式：预测正确数 / 总样本数
```

**示例**：
- 测试期间有 100 个样本
- 模型预测正确 55 个 → 准确率 = 55%
- 其中 20 个样本置信度 ≥ 0.65（产生交易信号）
- 这 20 笔交易中有 10 笔盈利 → 胜率 = 50%
