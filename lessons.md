## 一、核心警告

### 数据泄漏检测

| 准确率范围 | 判断 | 行动 |
|-----------|------|------|
| 52-55% | 正常 | 略优于随机 |
| 55-60% | 良好 | 业界正常水平 |
| 60-65% | 优秀 | 接近天花板 |
| **>65%** | **可疑** | 检查数据泄漏 |

### 模型可信度

| 模型 | 可信度 | 推荐 |
|------|--------|------|
| **CatBoost 20天** | ⭐⭐⭐⭐⭐ | **推荐** |
| CatBoost 5天 | ⭐⭐⭐ | 谨慎使用 |
| CatBoost 1天 | ⭐ | **不推荐**（过拟合） |
| LSTM/Transformer | ⭐ | **不推荐**（F1≈0） |

### 验证方法

| 方法 | 可信度 | 说明 |
|------|--------|------|
| **Walk-forward** | ✅ 唯一可信 | 每个fold重新训练 |
| 简单评估 | ❌ 不可信 | 结果虚高（83-90%） |

## 二、交易策略

### 异常策略（两年数据验证）

| 异常类型 | 策略 | 数据支持 |
|---------|------|---------|
| **IF high** | 🔴 减仓 | 5日收益-3.04%，胜率43% |
| **Z-Score + 当日下跌** | 🟢 抄底 | 5日收益+4.12%，**胜率72%** |

**关键洞察**：
- IF high = 系统性风险 → 减仓
- Z-Score + 当日下跌 = 超跌反弹 → 买入
- 下跌异常稀缺（24.1%），稀缺性增加价值

### 三周期一致预测策略 ⭐

| 模式 | 描述 | 20天准确率 | 策略 |
|------|------|-----------|------|
| **110** | 震荡回调 | **90%** | ⭐ 最优做空 |
| 000 | 一致看跌 | 79.19% | 强烈卖出 |
| 111 | 一致看涨 | 70.31% | 强烈买入 |

**最优策略**：震荡回调(110)做空，胜率90%，收益+4.25%

**市场环境影响**：
- 一致看涨：熊市胜率更高（71.43% vs 65.22%）
- 一致看跌：牛市准确率更高（81.63% vs 78.49%）

### 板块轮动策略

| 效应 | 数据支撑 | 可用性 |
|------|---------|--------|
| **周期/防御轮动** | 牛市周期占优74.3%，熊市防御占优65.9% | ⭐⭐⭐⭐ |
| **动量/反转** | 策略收益0.3%-0.5% | ⭐⭐ 不推荐 |

**实盘建议**：
1. 判断 HSI 20天收益 → 确定牛/熊/震荡
2. 牛市配周期（半导体、科技、消费），熊市配防御（保险、公用事业、银行）
3. **不要**根据动量/反转信号单独交易
4. **领涨频率高≠策略收益高**

### 加密货币策略

**股票策略不适用于加密货币**：

| 策略 | 股票 | 加密货币 | 建议 |
|------|------|---------|------|
| IF high → 减仓 | ✅ 有效 | ❌ 无效 | **不使用** |
| Z-Score抄底 | ✅ 推荐 | ⚠️ 高风险 | **禁止** |

## 三、开发规范

### 数据泄漏防护

```python
# ❌ 错误：使用当日数据
df['Feature'] = df['Close'].pct_change()

# ✅ 正确：使用昨日数据
df['Feature'] = df['Close'].pct_change().shift(1)
```

**高风险特征**（必须 `.shift(1)`）：
- 所有 `.rolling()` 计算的特征
- BB_Position、Price_Percentile
- 动量分析中的 `future_return`

## 四、常见错误与修复

### 1. 动量分析数据泄漏

**问题**：`future_return` 没有正确 shift

```python
# ❌ 错误
future_return = returns.rolling(5).sum()

# ✅ 正确
future_return = returns.rolling(5).sum().shift(-5)
```

**教训**：简单相关性分析可能存在数据泄漏，应以 Walk-forward 验证为准

### 2. 回撤计算错误（固定持有期策略）

**问题**：时间序列回撤对固定持有期策略是错误的

**修复**：使用批次回撤计算

```python
# ✅ 正确：每个信号日期视为独立投资批次
batches = trades.groupby(trades.index).agg({
    'strategy_return': 'mean'
})
cumulative_values = np.cumsum(batch_returns * batch_weights) + 1.0
```

**结果**：最大回撤从 -84% 修正为 -0.55%

### 3. 夏普比率年化错误

**问题**：假设一年可以做 (252/horizon) 次独立交易

**修复**：使用月度收益年化

```python
# ✅ 正确
annualized_return = monthly_return * 12
annualized_std = batch_std * np.sqrt(12)
```

**结果**：夏普比率从 0.29 修正为 0.97

### 4. Walk-forward 样本量不足

| 训练样本数 | 准确率 | 评估 |
|-----------|--------|------|
| <200 | 38% | 不可信 |
| ≥200 | 55-65% | 稳定 |

**教训**：训练样本不足时结果不可信
