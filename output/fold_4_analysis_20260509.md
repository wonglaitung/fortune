## 三、准确率 vs 命中率 vs 胜率 的区别

### 三个指标的定义

| 指标 | 定义 | 计算公式 | Fold 4 数值 |
|------|------|----------|-------------|
| **准确率** | 预测方向正确的比例 | (预测=实际方向) / 总样本 | **37.5%** |
| **命中率** | 实际收益为正的比例 | (收益>0) / 该区间样本 | 各区间不同 |
| **胜率** | 交易后盈利的比例 | (交易净收益>0) / 交易次数 | **97.75%** |

### 三者的关系

```
准确率 (37.5%) ──────────────────────────────────────────── 低
    │
    │  风险过滤 (置信度>=0.55, ATR过滤, 震荡市过滤)
    │
    ↓
胜率 (97.75%) ─────────────────────────────────────────── 高
```

**关键发现**:
- 准确率 37.5% = 模型方向判断能力差
- 命中率 100% (高置信区间) = 这些样本实际盈利
- 胜率 97.75% = 交易后 97.75% 盈利

## 四、异常现象解释

### 4.1 为什么准确率低但胜率高？

| 步骤 | 说明 |
|------|------|
| **原始样本** | 1064 个样本，准确率 37.5% |
| **风险过滤** | 过滤掉 ~975 个样本 |
| **交易样本** | 89 笔交易，胜率 97.75% |

**核心原因**:
1. 模型预测方向错误 (准确率 37.5%)
2. 但风险过滤恰好过滤掉了大部分错误预测
3. 剩下的 89 笔交易恰好预测正确

### 4.2 交易信号过滤流程

```
1064 个样本
    │
    ├─ 置信度 >= 0.55 ──────────── 过滤 ~700 个样本
    │
    ├─ ATR_Risk_Score <= 0.8 ──── 过滤高波动期
    │
    └─ 震荡市需概率 >= 0.75 ───── 过滤震荡市低置信信号
    │
    ↓
89 笔交易 (8.4%)
    │
    └─ 其中 87 笔盈利 (胜率 97.75%)
```

## 五、风险提示

### ⚠️ 不可持续的因素

| 风险 | 说明 | 严重程度 |
|------|------|----------|
| **样本太少** | 89 笔交易，统计意义不足 | 🔴 高 |
| **100% 命中率** | 极端情况，不可复制 | 🔴 高 |
| **准确率 37.5%** | 模型方向判断能力差 | 🔴 高 |
| **风险过滤巧合** | 历史有效，未来不一定 | 🟡 中 |

### ⚠️ 过拟合风险

```
问题: 准确率 37.5%，但风险过滤后胜率 97.75%

风险:
- 风险过滤可能"恰好"过滤掉了错误预测
- 这在历史回测中有效
- 未来实盘不一定有效
```

## 八、结论

### 综合评估: ⚠️ 需警惕

| 方面 | 评估 |
|------|------|
| **高置信度信号** | ✅ 命中率 100%，但样本少 |
