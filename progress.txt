### 综合分析股票预测概率修复 ⭐ 重要修复

**问题**：农行推荐理由显示的 CatBoost 概率（0.53）是 1 天预测概率，而不是 20 天预测概率（0.15）。

**原因分析**：
- 传给大模型的表格包含三周期概率（1天、5天、20天三列）
- 大模型在生成推荐理由时混淆了不同周期的概率
- 农行数据：20天预测概率=0.15（下跌），1天预测概率=0.53（上涨）

**解决方案**：
- 函数返回两个版本的表格：
  - `ensemble`：只包含20天概率，用于大模型决策
  - `ensemble_email`：包含三周期预测，用于邮件显示
- 大模型决策时只看到20天概率，避免混淆
- 邮件保留完整三周期预测表格，用户可查看完整信息

**改动文件**：
- `comprehensive_analysis.py:695-930` - `extract_ml_predictions()` 返回两个版本

**输出对比**：
| 用途 | 格式 | 说明 |
|------|------|------|
| 大模型决策 | `| 预测方向 \| 上涨概率 \|` | 只有20天概率 |
| 邮件显示 | `| 1天预测 \| 5天预测 \| 20天预测 \| 筹码阻力 \|` | 三周期+筹码 |

### 市场状态计算修复 ⭐ 重要修复

### 特征缓存机制 ⭐ 新增

**问题**：三周期训练/预测时，特征重复计算3次

**解决方案**：添加特征级缓存

**缓存机制**：
| 项目 | 说明 |
|------|------|
| 缓存位置 | `data/feature_cache/` |
| 缓存键 | 股票代码 + 数据最后日期 |
| 有效期 | 7天（与原始数据缓存一致） |
| 格式 | pickle |

**性能对比**：
| 场景 | 耗时 | 加速 |
|------|------|------|
| 无缓存 | 54s | 1x |
| 有缓存 | 0.32s | **170x** |

**使用方法**：
```python
# 训练时自动使用缓存
model.train(codes, horizon=20, use_feature_cache=True)  # 默认开启

# 预测时自动使用缓存
model.predict('0005.HK', use_feature_cache=True)  # 默认开启
```

**注意事项**：
- 缓存不含标签（标签依赖于 horizon）
- 三周期训练时，1d/5d/20d 共用同一份特征缓存

### 一、三周期预测验证 ⭐ 最重要发现

#### 恒生指数

| 周期 | 准确率 | 策略建议 |
|------|--------|---------|
| 1天 | 51.31% | 噪音大，仅供参考 |
| 5天 | 57.17% | 趋势确认，辅助判断 |
| **20天** | **80.73%** | **最可靠，主要决策依据** |

**最优策略**：震荡回调模式(110)做空，胜率 **90%**，收益 **+4.25%**

**八大模式准确率**：
| 模式 | 描述 | 20天准确率 | 策略 |

| 111 | 一致看涨 | 70.31% | 强烈买入 |

### 四、板块轮动法则

**核心策略**：
```
轮动策略 = 周期/防御切换（有效）
         ≠ 动量/反转策略（无效）
         ≠ 领涨频率追逐（不可靠）
```

| 市场状态 | 配置 | 仓位 |
|---------|------|------|
| 牛市（HSI 20日收益>75%分位） | 半导体30%、生物医药25%、科技20%、消费15% | 90% |
| 熊市（HSI 20日收益<25%分位） | 保险30%、公用事业25%、银行20% | 75% |
| 震荡 | 动态选择相对强势板块 | 60% |

## 快速参考

### 核心警告

1. **数据泄漏**：准确率 > 65% 通常是数据泄漏信号
2. **CatBoost 1天**：严重过拟合，不推荐使用
3. **Walk-forward**：唯一可信的验证方法
4. **动量策略**：效果有限，不推荐单独使用

### 常用命令

```bash
# 恒指预测
python3 hsi_prediction.py --no-email

# 综合分析
python3 comprehensive_analysis.py

# 异常检测
python3 detect_stock_anomalies.py --mode standalone --mode-type deep

# Walk-forward 验证
python3 ml_services/walk_forward_validation.py --model-type catboost --horizon 20

# 板块轮动验证
python3 verify_sector_rotation.py

# 个股三周期验证（新增）

python3 ml_services/stock_three_horizon.py --sector bank    # 板块
python3 ml_services/stock_three_horizon.py --all            # 全量
```
